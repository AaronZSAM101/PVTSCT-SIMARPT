## 标牌使用说明

### EuroCAT-X-TAG
基于上海终端区的空管一体化自动系统以及培训系统标牌开发的供EuroScope使用的标牌
#### 使用前说明
**1.该系列标牌基于上海终端管制区的空管一体化系统以及莱斯空管一体化模拟系统，并结合模拟飞行的实际需要进行的配置，如有客制化需求，请先结合ES阅读完本部分，将原本的标牌备份后再根据自己的需要进行修改**

**2.该系列标牌基于Easy Mode下配置，不对S-Mode模式进行保障，欢迎乐意保障的人来保障这个S-Mode**

**3.如你想保留本标牌的欧洲猫样式，请保留所有Static String的空格**

**4.搭配一分钟预测线、公制单位以及EuroCAT-X HMI配色食用效果最佳**

**5.EuroCAT-X SweatBox的标牌结构与EuroCAT-X Shanghai TMA标牌结构相同，功能差异仅为开盒子时高度可以常显示米制高度**
#### Item说明
| Item | 功能 | 使用插件 |
| :---: | --- | --- |
| 分配跑道 | 左键选跑道，右键选程序 | ES原生 |
| 警报 | RAM CLAM DUPE...| ES原生 |
| 相似呼号指示 | 在算法识别到有相似呼号时显示SC，左键点击后显示关联呼号，点击关联呼号即可触发```.find 该机组呼号```的效果 | MTEPlugin |
| 通讯方式指示 | 如机组计划写/v会空白，可在尾流等级后面空白处点击改变通讯类别。| ES原生 |
| 呼号 | 左键移交，右键高亮 | ES原生、Topsky |
| 当前高度 | 单位为米，搭配```EuroScope_Thales_Topsky.ttf```的字体文件，```Plugin_lined.txt```的插件引导文件以及```MTEPTransLvl.csv```文件使用可使当前高度在低于TL时显示下划线 | MTEPlugin |
| 上下高指示 | 指示上下高 | MTEPlugin |
| 分配高度 | 真实情况下是左键选高度中键确认，但由于每个人鼠标不尽相同，以及在ES中使用中键点出来的功能有可能是左键也有可能是右键，故设定为**左右键均可选择高度**，选择后分配高度变白，确认机组复诵正确后再次点击即可变回原色 | MTEPlugin |
| 地速显示、下降率显示 | 地速单位KMH，如420KMH，则显示042；下降率显示为FPM，以四位数字显示 | MTEPlugin |
| 目的地机场 | 左键开飞行计划（ES），右键显示机组RTE线（Topsky）| ES原生、Topsky |
| 指令航向 | 建议直接使用Topsky给航向的方式（点雷达目标再拉线），也可直接输入“Hxxx” | ES原生、Topsky |
| 指令速度 | 该项目在标牌中无点击项目，建议通常的操作方法是直接在自由文本栏中输入“Nxxx”（例：N180）| ES原生 |
| 自由文本 | 若有指令航向的情况，请在指令航向的右侧输入你的自由文本 | ES原生 |
| 给定高度 | 由于Sweatbox只能下英制高度的特性，故直接选择对应米制向上取整后的英制高度即可，学员端则显示该英制高度的米制高度（不为整数）| ES原生 |
| 给定航向/直飞 | 左键长按AHDG，会拉出一条线，拉至对应航向后松开，机组即可朝这个方向转向（该操作同时会影响学员端分配航向的显示）；右键点击AHDG即可操作飞机直飞某个点 | ES原生 |

### EuroCAT-X Shanghai TMA标牌结构

第一行：分配跑道* 警报* 相似呼号指示* 重连指示* 通讯方式指示*

第二行：呼号* 尾流等级*

第三行：当前高度* 上下高指示* 分配高度* 地速显示 下降率显示（平飞时不显示）

第四行：目的地机场* 机型

第五行：指令航向* 指令速度* 自由文本*
